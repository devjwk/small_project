HTML Tags Reference

Basic HTML

Tag Name Description
<!DOCTYPE> Doctype declaration Defines the document type
<html> html represents the root of an HTML document.
<head> head Contains metadata/information for the document
<title> title Defines a title for the document
<body> body Defines the document's body
<h1> to <h6> headings Defines HTML headings
<p> paragraph Defines a paragraph
<br> break Inserts a single line break
<hr> thematic break Defines a thematic change in the content
<!--..--> comments Defines a comment

Formatting

Tag Name Description
<abbr> acronym Defines an abbreviation or an acronym
<address> address Defines contact information for the author/owner of a document/article
<b> bold Defines bold text
<bdi> Bi-Directional Isolation Isolates a part of text that might be formatted in a different direction from other text outside it
<bdo> Bi-Directionala Override Overrides the current text direction
<blockquote> blockquote Defines a section that is quoted from another source
<cite> cite Defines the title of a work
<code> code Defines a piece of computer code
<del> deleted code Defines text that has been deleted from a document
<dfn> defined term Specifies a term that is going to be defined within the content
<em> emphasize Specifies a term that is going to be defined within the content
<i> italic Defines a part of text in an alternate voice or mood
<ins> inserted Defines a text that has been inserted into a document
<kbd> keyboard Defines keyboard input
<mark> mark Defines marked/highlited text
<meter> meter Defines a scalar measurement within a known range(a gauge)
<pre> preformat Defines preformatted text
<progress> progress Represents the progress of a tesk
<q> quotation Defines a short quotation
<rt> Defines an explanation/pronunciation of characters(for East Asian typography)
<ruby> ruby annotation Defines a ruby annotation(for East Asian typography)
<s> Defines text that is no longer correct
<samp> sample Defines sample output from a computer program
<small> small Defines smaller text
<strong> strong Defines important text
<sub> subscript Defines subscripted text
<sup> superscript Defines superscripted text
<template> template Defines a container for content that should be hidden when the page loads
<time> time Defines a specific time (or datetime)
<u> unqrticulate Defines some text that is unarticulated and styled differently from normal text
<var> variables Defines a variable
<wbr> Word Break Oppurtunity Defines a possible line-break
